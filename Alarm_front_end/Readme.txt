
Recommended IDE – NetBeans IDE

Steps to follow : 

1. Download Alarm_front_end and extract it to somewhere.

2. Go to \build\classes folder.

3. Open Command Prompt from there.

4. Type ‘start rmiregistry’.

5. Then open AlarmRestAPI project with IntelliJ or similar IDE.

6. Find AlarmRestApiApplication.java and run it.

7. Make sure to connect to the internet.

8. After that open Alarm_front_end project from NetBeans IDE.

9. Right click and run the Server.java file.

10 .Right click and run the Login.java file.
